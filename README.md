# 🔐 Login & Registration System

A simple Login and Registration system built using **HTML, CSS, and JavaScript**. This project demonstrates the fundamentals of user authentication interfaces, form handling, and frontend validation.

## 🚀 Features

* User Registration Form
* User Login Form
* Clean and Responsive UI
* Basic Form Validation
* Easy-to-understand Code Structure

## 🛠️ Technologies Used

* HTML
* CSS
